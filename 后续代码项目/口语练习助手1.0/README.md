# 口语练习助手 1.0

这是“英语单词跟读与 AI 情景对话练习助手”的免费演示版程序。

## 当前功能

- 单词列表和单词卡片
- 浏览器标准英文发音
- 麦克风录音和录音回放
- 浏览器语音识别评分
- “会读了 / 需复习”标记
- 本地错词本
- 模板式情景对话
- 本地练习记录
- 学生姓名记录
- CSV 导出评分记录

## 使用方式

直接用浏览器打开 `index.html`。

推荐使用 Chrome 或 Edge。录音功能需要允许浏览器访问麦克风。

朗读评分功能依赖浏览器语音识别能力。它会把学生朗读识别成英文文本，再和目标单词做相似度比较，适合作为免费版练习参考，不等同于专业语音评测接口。

点击“朗读评分”后，请在 6 秒内读出当前单词。读完后系统会自动结束；如果一直在听，也可以再次点击按钮手动停止。

## 费用说明

当前版本不依赖服务器、不依赖收费接口，使用浏览器自带能力和本地存储，演示成本为 0 元。
