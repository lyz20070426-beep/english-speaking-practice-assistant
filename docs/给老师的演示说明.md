# 英语单词跟读与 AI 情景对话练习助手 1.0 演示说明

## 一、项目定位

本项目第一版先做一个轻量、免费、容易上手的英语口语练习网页工具，不做复杂平台，也不依赖收费接口。

第一版重点解决两个问题：

1. 学生可以围绕课程单词进行标准发音跟读。
2. 学生可以围绕校园、课堂、课文主题进行简单英语情景对话。

## 二、当前演示版已经实现的功能

### 1. 单词跟读

当前已经录入第一批试点单词，共 43 个。

学生可以：

- 查看单词、音标、中文释义和例句。
- 点击按钮听浏览器标准英文发音。
- 点击录音按钮进行跟读。
- 回放自己的录音。
- 标记“会读了”或“需复习”。

### 2. 错词本

学生把不熟悉或读不准的单词标记为“需复习”后，系统会自动汇总到错词本，方便后续反复练习。

### 3. 情景对话

当前演示版内置 4 个情景：

- 自我介绍
- 校园生活
- 课堂问答
- 人物采访

学生输入英文回答后，系统会根据场景继续追问，并给出简单练习建议。

说明：当前 1.0 演示版采用模板式对话，主要用于验证练习流程。后续如果需要更自然的 AI 对话，可以再接入大模型接口。

### 4. 练习记录

系统会在当前浏览器中保存本地练习记录，包括：

- 已练单词数量
- 需复习单词数量
- 情景对话次数
- 错词本列表

## 四、使用方式

当前版本是本地网页演示版。

打开方式：

1. 使用 Chrome 或 Edge 浏览器。
2. 打开程序目录中的 `index.html`。
3. 录音时允许浏览器访问麦克风。

当前程序位置：

```text
后续代码项目/口语练习助手1.0/index.html
```

## 五、适合演示的流程

建议演示时按以下顺序：

1. 打开网页，进入“单词跟读”。
2. 选择一个单词，例如 `automobile`。
3. 点击标准发音，听浏览器朗读。
4. 点击开始录音，学生跟读。
5. 点击回放录音，对比自己读音。
6. 标记“需复习”，查看错词本。
7. 切换到“情景对话”，选择一个场景。
8. 输入英文回答，查看系统追问和建议。
9. 切换到“练习记录”，查看本地记录。

## 六、当前版本限制

当前版本是 1.0 免费演示版，所以暂不包含：

- 专业发音评分
- 音素级纠错
- 教师后台
- 班级管理
- 学生账号系统
- 云端数据同步
- 真实大模型对话接口

这些功能可以作为后续升级方向。
