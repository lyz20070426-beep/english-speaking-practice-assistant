# 英语口语练习助手

## 项目简介

本项目计划开发一个面向学生的英语口语练习助手，用于帮助学生进行英语单词、句子、对话和考试场景下的口语练习。系统目标是降低学生开口练习英语的门槛，让学生可以在课后、家中或学校机房中反复练习，并获得及时反馈。

项目当前处于需求整理和技术方案准备阶段。

## 项目目标

- 支持学生进行英语口语跟读、朗读和对话练习。
- 根据学校教学内容整理练习素材。
- 记录学生练习情况，方便后续复盘。
- 为老师和家长提供阶段性学习反馈。
- 逐步形成可在学校环境中使用的英语口语训练系统。

## 推荐开发流程

1. 先整理学校真实需求和使用场景。
2. 根据单词表、老师反馈和学生练习场景确定第一版功能。
3. 设计最小可用版本，优先做口语练习核心流程。
4. 开发前端页面、后端接口和数据存储。
5. 让学生试用，收集反馈后迭代。

## 1.0 版范围

第一版先做一个免费优先、轻量可演示的网页工具：

- 单词跟读：标准发音、学生录音、录音回放、需复习标记。
- 情景对话：围绕校园、课堂、课文主题进行模板式英语对话。
- 练习记录：记录练习次数、需复习单词和对话次数。

暂不承诺专业发音评分、复杂教师后台和大规模班级管理。

## 协作方式
